## 🚀 **구현한 주요 기능들**

### 🗺️ **실시간 위치 기반 서비스**
> **"내 주변 주유소를 지도로 보여줘!"**

**핵심 기능:**
- **GPS 위치 자동 감지** 및 현재 위치 표시
- **Google Maps API** 연동으로 실제 지도 서비스 구현
- **반경 3km 내 주유소** 자동 검색 및 마커 표시
- **거리와 평점을 고려한 추천 알고리즘** (나름대로!)

### 📊 **유가 예측 시스템**
> **"내일 주유하는 게 좋을까? 오늘 하는 게 좋을까?"**

**신기한 기능들:**
- **7일간 유가 데이터** 기반 다음 날 가격 예측
- **Chart.js**로 깔끔한 그래프 시각화
- **"💡 오늘 주유해야해용~" / "💡 내일 넣는게 조타잉~"** 귀여운 추천 메시지
- **신뢰도 표시**까지 구현 (초보치고 나름 꼼꼼!)

## 🎓 **이 프로젝트를 통해 배운 것들**

### 💻 **웹개발의 기초 체험**
**"HTML, CSS, JavaScript가 어떻게 작동하는지 몸으로 체득!"**

1. **HTML 구조의 중요성**: 900줄짜리 파일을 만들면서 구조의 필요성을 절실히 깨달음
2. **CSS 스타일링의 재미**: 반응형 웹을 만들면서 미디어쿼리의 신기함 경험
3. **JavaScript의 역동성**: API 연동과 DOM 조작으로 동적 웹의 매력 발견

### 🗺️ **API 연동의 신세계**
**"외부 서비스와 연결하면 이렇게 멋진 게 만들어져?"**

- **Google Maps API**: 지도 서비스 연동의 복잡함과 성취감
- **공공 데이터 활용**: 실제 유가 정보를 가져와서 활용하는 뿌듯함
- **CORS 문제**: 처음 마주친 웹개발의 현실적 장벽들

### 🎯 **사용자 경험(UX)에 대한 첫 고민**
- **"사용자가 정말로 필요한 정보가 뭘까?"**
- 복잡한 데이터를 **직관적으로 보여주는 방법** 고민
- **모바일 사용자**를 고려한 반응형 디자인의 중요성
